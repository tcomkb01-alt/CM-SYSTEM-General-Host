# CM System (Classroom Management System)

ระบบจัดการชั้นเรียนอัจฉริยะแบบครบวงจร พัฒนาด้วย PHP MVC สำหรับครูผู้สอนเพื่อจัดการข้อมูลนักเรียน การเช็คชื่อ และการมอบหมายงานอย่างมีประสิทธิภาพ

## 🚀 ความสามารถของระบบ (Key Features)

### 1. การจัดการข้อมูลพื้นฐาน
*   **ระบบจัดการนักเรียน (Student Management):** เพิ่ม, แก้ไข, ลบ และนำเข้าข้อมูลนักเรียนผ่านไฟล์ CSV
*   **ระบบจัดการห้องเรียน (Classroom Management):** สร้างห้องเรียนรายวิชา, กำหนดรหัสห้องเรียน 6 หลัก, และระบบเกณฑ์ผ่านการเข้าเรียน
*   **โปรไฟล์ผู้ใช้ (Admin Profile):** จัดการข้อมูลส่วนตัว และเปลี่ยนภาพโปรไฟล์แอดมิน

### 2. ระบบเช็คชื่อเข้าเรียน (Attendance System)
*   **การเช็คชื่อหลายรูปแบบ:** รองรับการเช็คชื่อแบบ Manual, การยิงบาร์โค้ด (Barcode Scanner), และการสแกน QR Code
*   **Student Portal:** นักเรียนสามารถสแกน QR Code เพื่อเช็คชื่อด้วยตัวเองผ่านอุปกรณ์มือถือ
*   **ประวัติการเช็คชื่อ:** ตรวจสอบย้อนหลังและสรุปสถิติการมาเรียน (มา, สาย, ขาด, ลา) เป็นเปอร์เซ็นต์

### 3. ระบบจัดการงานและคะแนน (Assignment & Grading)
*   **การมอบหมายงาน:** สร้างหัวข้อการบ้าน กำหนดคะแนนเต็มและวันครบกำหนด
*   **ระบบให้คะแนน:** ครูสามารถให้คะแนนและติดตามสถานะการส่งงานของนักเรียนแต่ละคนได้
*   **ภาพรวมคะแนน:** สรุปคะแนนสะสมและสถานะการส่งงานในภาพรวมห้องเรียน

### 4. ฟีเจอร์เสริมอัจฉริยะ
*   **Student Card Generator:** ระบบสร้างบัตรนักเรียนพร้อม QR Code และ Barcode ในตัว สามารถสั่งพิมพ์เป็นไฟล์ PDF ได้ทันที
*   **Report System:** ออกรายงานสรุปผลการเข้าเรียนและคะแนนเก็บ
*   **License System:** ระบบล็อคโดเมนและการตรวจสอบสิทธิ์การใช้งาน (Software Licensing)

## 🗄️ โครงสร้างฐานข้อมูล (Database Schema)

ระบบใช้ฐานข้อมูล MySQL/MariaDB โดยมีตารางหลักดังนี้:

*   **users:** เก็บข้อมูลผู้ใช้งาน (Admin/Teacher) และรหัสผ่านที่เข้ารหัส
*   **students:** เก็บข้อมูลนักเรียน, รหัสประจำตัว, และข้อมูล Barcode
*   **classrooms:** เก็บข้อมูลห้องเรียนและวิชา
*   **classroom_students:** ตารางกลางเชื่อมความสัมพันธ์นักเรียนกับห้องเรียน
*   **attendance_sessions:** เก็บข้อมูลรอบการเช็คชื่อ (วันที่, คาบเรียน)
*   **attendance_records:** เก็บผลการเช็คชื่อรายบุคคล
*   **assignments / assignment_scores:** เก็บข้อมูลงานและคะแนนเก็บ
*   **school_settings:** ตั้งค่าข้อมูลโรงเรียนและโลโก้
*   **activity_logs / login_logs:** บันทึกประวัติการใช้งานและประวัติการเข้าระบบ

## 🔒 ระบบความปลอดภัย (Security Features)

1.  **Authentication & Authorization:** ระบบตรวจสอบสิทธิ์เข้าใช้งาน และแบ่งระดับ Role ชัดเจน
2.  **CSRF Protection:** ป้องกันการโจมตีแบบ Cross-Site Request Forgery ด้วยระบบ Token ในทุกฟอร์ม
3.  **Password Hashing:** เข้ารหัสรหัสผ่านด้วย `BCRYPT` มาตรฐานสูง
4.  **SQL Injection Prevention:** ใช้งาน Prepared Statements และ Parameterized Queries ผ่าน PDO Core
5.  **Secure Headers:** มีการส่ง Security Headers เช่น `X-Frame-Options`, `X-XSS-Protection`
6.  **License Management:** ระบบ Domain Lock และ License Validation ป้องกันการใช้งานโดยไม่ได้รับอนุญาต
7.  **Rate Limiting & Logging:** บันทึก Login Logs และมีระบบล็อคบัญชีเมื่อกรอกรหัสผิดพลาดหลายครั้ง
